# 터미널 1:1 채팅 사용법 (Windows)

간단한 터미널 채팅 프로그램이에요. 메시지는 **끝에서 끝까지 암호화**(E2E) 되어서
서버도 내용을 못 봐요. 아래 순서대로 한 번만 세팅하면 돼요.

---

## 1단계. Node.js 설치 (최초 1회)

1. https://nodejs.org/ 접속
2. **LTS 버전** (왼쪽 녹색 버튼) 다운로드
3. 다운받은 설치 파일 실행 → 전부 **Next** → **Install**
4. 설치 완료

**설치 확인**:
- 윈도우 검색창에 `PowerShell` 입력 → 실행
- 아래 명령어 복사해서 붙여넣고 Enter:
  ```powershell
  node -v
  ```
- `v20.x.x` 같은 숫자가 나오면 OK. 안 나오면 컴퓨터 재시작 후 다시 시도.

---

## 2단계. 채팅 파일 준비 (최초 1회)

1. **받은 zip 파일** (`chat-cli.zip`)을 **바탕화면**에 저장
2. 우클릭 → **압축 풀기** → 폴더 생김 (예: `chat-cli`)
3. PowerShell 에서 그 폴더로 이동:
   ```powershell
   cd $HOME\Desktop\chat-cli
   ```
4. 필요한 것 설치 (1회만):
   ```powershell
   npm install
   ```
   → 몇 초 걸림. 끝나면 다음 줄 커서 깜빡임.

---

## 3단계. 채팅하기 (매번)

### 방법 A. 상대가 링크를 보냈을 때 (참여)

1. PowerShell 열기
2. 폴더로 이동:
   ```powershell
   cd $HOME\Desktop\chat-cli
   ```
3. 받은 **초대링크**를 따옴표로 감싸서 입력:
   ```powershell
   node chat.js "wss://chat-cli-7woy.onrender.com#여기에받은토큰"
   ```
   ※ 따옴표 `""` 꼭 붙여야 해요. `#` 때문에 없으면 오류.

4. `✓ 연결됨` 뜨면 입력 시작. 엔터 치면 전송.

### 방법 B. 내가 방 만들기

1. PowerShell → 폴더 이동:
   ```powershell
   cd $HOME\Desktop\chat-cli
   ```
2. 실행:
   ```powershell
   node chat.js
   ```
3. 화면에 **초대링크** 한 줄이 뜸 → **그걸 상대에게 전달**
4. 상대가 접속하면 `✓ 연결됨` 자동으로 뜸 → 대화 시작

### 종료

- **Ctrl + C** 누르기

## 자주 겪는 문제

| 증상 | 해결 |
|------|------|
| `node : 용어가 인식되지 않습니다` | Node 설치 안 됐거나 재시작 안 한 상태. 컴퓨터 재부팅. |
| `Cannot find module 'ws'` | `npm install` 안 함. 2단계 4번 실행. |
| 첫 연결이 30~60초 걸림 | 정상. 서버가 자고 있다가 깨는 중. 기다리면 연결됨. |
| `room full` | 방에 이미 2명 있음. 새 링크 받아야 함. |
| 연결 끊김 / 멈춤 | Ctrl+C 로 끄고 다시 실행. 초대링크 재사용 가능. |

---

## 참고

- **초대링크는 1회성**: 종료하면 새로 만들어야 함 (B가 매번 새 링크 생성)
- **히스토리 저장 안 됨**: 창 닫으면 내용 사라짐
- **한 방에 2명만**: 제3자 접속 자동 차단
- **네트워크**: 회사 방화벽 상관없이 HTTPS 로 통신해서 대부분 통과
